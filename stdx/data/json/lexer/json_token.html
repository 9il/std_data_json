    Struct JSONToken

    A low-level JSON token as returned by JSONLexer.

    Fields

    Name Type Description
    location Location The location of the token in the input.

    Properties

    Name Type Description
    boolean [get, set] bool Gets/sets the boolean value of the token.
    kind [get, set] JSONToken.Kind Gets/sets the kind of the represented token.
    number [get, set] JSONNumber Gets/sets the numeric value of the token.
    string [get, set] JSONString Gets/sets the string value of the token.

    Methods

    Name Description
    opEquals Enables equality comparisons.
    toHash Enables usage of JSONToken as an associative array key.
    toString Converts the token to a string representation.

    Name Description
    Kind The kind of token represented.
