    Struct JSONValue

    Represents a generic JSON value.

    The JSONValue type is based on std.variant.Algebraic and as such provides the usual binary and unary operators for handling the contained raw value.

    Raw values can be either null, bool, double, string, JSONValue[] or JSONValue[string].

    Example

    Shows the basic construction and operations on JSON values. 

    JSONValue a = 12;
JSONValue b = 13;

assert(a == 12.0);
assert(b == 13.0);
//assert(a + b == 25.0);

auto c = JSONValue([a, b]);
assert(c.get!(JSONValue[])[0] == 12.0);
assert(c.get!(JSONValue[])[1] == 13.0);
assert(c[0] == 12L);
assert(c[1] == 13L);

auto d = JSONValue(["a": a, "b": b]);
assert(d.get!(JSONValue[string])["a"] == 12.0);
assert(d.get!(JSONValue[string])["b"] == 13.0);
assert(d["a"] == 12L);
assert(d["b"] == 13L);

    Example

    Using opt to quickly access a descendant value. 

    JSONValue subobj = ["b": JSONValue(1.0), "c": JSONValue(2.0)];
JSONValue obj = ["a": subobj];

assert(obj.opt("x").isNull);
assert(obj.opt("a", "b") == 1.0);
assert(obj.opt("a", "c") == 2.0);
assert(obj.opt("a", "x").isNull);
